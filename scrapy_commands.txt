1) Show Scrapy commands
$ scrapy

2) Crate a new project
$ scrapy startproject <spidername>

3) Scrapy shell
$ scrapy shell http://quotes.toscrape.com/

4) Genspider
$ cd <project dir>
$ scrapy genspider <spyder name> <domain to scrape>
# Example: $ scrapy genspider quotes quotes.toscrape.com
# Generate new spider using pre-defined templates

5) List all available spiders
$ scrapy list

6) Start Crawling
$ scrapy crawl <spider name>

7) Requests Commands
$ response.xpath('//h1/a/text()').extract()
$ response.xpath('//*[@class="tag-item"]/a/text()').extract()
$ response.xpath('//*[@class="next"]/a/@href').extract_first()
